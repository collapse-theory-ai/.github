# AGENTS.md

Canonical instructions for AI coding agents working in this organization-default
repository.

## Scope

- This repo stores shared GitHub defaults and reusable templates for the
  `collapse-theory-ai` organization.
- Repository-local `AGENTS.md` files override these defaults.
- Keep secrets, credentials, and generated runtime state out of this public repo.

## Standards

- Use conventional branch prefixes such as `feat/`, `fix/`, `docs/`, `ci/`, and
  `chore/`.
- Use Conventional Commit subjects for commits and pull request titles.
- Prefer small, reviewable changes.
- Verify templates and workflows before claiming completion.
